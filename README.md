\# Hello Node — Railway \& Netlify Deployment



Este proyecto contiene un ejemplo simple de aplicación \*\*Node.js con Express\*\* desplegada en \*\*Railway\*\*, y una página \*\*estática HTML\*\* desplegada en \*\*Netlify\*\*.



\## Herramientas utilizadas



\- \*\*Node.js\*\* + \*\*Express\*\* (para el servidor dinámico)

\- \*\*HTML5 / CSS básico\*\* (para el sitio estático)

\- \*\*Railway\*\* (para el deploy del servidor Node.js)

\- \*\*Netlify\*\* (para el deploy del sitio estático)
